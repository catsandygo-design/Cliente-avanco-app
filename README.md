# Aplicativo CRM Crédito — Node.js + React

## Rodar backend
```bash
cd backend
npm install
npm run dev
```
API: http://localhost:3001

## Rodar frontend
```bash
cd frontend
npm install
npm run dev
```
Tela: http://localhost:5173

## O que já tem
- Cadastro de cliente/reserva
- Filtro por imobiliária e pesquisa
- Etapas do processo
- Avanço em barra
- Prioridade por bolinha: verde, amarela e vermelha
- Observações e status editáveis
- Resumo geral

## Próximo passo recomendado
Trocar a memória temporária do backend por banco PostgreSQL/Supabase.
